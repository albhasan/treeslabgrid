# Add the list's names to the data frames in the list

Given a list of data frames, add each list's name to its corresponding
data frame.

## Usage

``` r
listnames2dataframes(df_ls, cname)
```

## Arguments

- df_ls:

  a named list of data frames.

- cname:

  a character(1). The name for the new column.

## Value

        a list of data frames.
